== Description ==

= WP PayMobile is a new monetization method for your website. Get payments by SMS / Phone Call for revealing content. =

* Monetize your website by taking mobile micropayments
* Uses PayGol micropayments gateway : completely free to signup and no monthly fees applied
* Easy to configure the message your visitors will see to encourage them to convert into payers
* Easy to configure the button they will click : custom, standard red,blue,yellow(default)

== Installation ==

1. Upload `WP-PayMobile` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. The shortcode contains a start tag [wp_paymobile_popup] and an end tag [/wp_paymobile_popup].
The content between the shortcode will not be shown unless they pay by mobile. 
Example: [wp_paymobile_popup] this content will be shown to paying visitors [/wp_paymobile_popup]

For detailed installation instructions open "documentation.html" with any browser (Chrome/FF/etc) inside the plugin folder.
