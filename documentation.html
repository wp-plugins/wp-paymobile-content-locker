A) Installing the plugin - top

Installing the plugin is very easy just like any other wordpress plugin.
Connect to your hosting and upload the extracted zip file folder of the plugin into wp-content/plugins/
Then go to wp-admin->plugins->and activate WP PayMobile

B) Setting Plugin Options - top

The first thing you might want to do is to setup WP PayMobile Options :
Into wordpress admin, go to the newly installed plugin section called "WP Paymobile".

  • You can choose if you want to use a custom button/image or the default red, blue or yellow one (default is yellow).
  • You can setup your own message to encourage users to Pay with their Mobile to see all the page/post content.

C) Adding Shortcode - top

Adding the shortcode is a simple straight forward job. You can use it on as many pages/posts as you want.
The shortcode contains a start tag [wp_paymobile_popup] and an end tag [/wp_paymobile_popup].
The content between the shortcode will not be shown unless they pay by mobile.
Example: [wp_paymobile_popup] this content will be shown to paying visitors [/wp_paymobile_popup]

D) Configuring PayGol Payment Gateway - top

You are required to register for FREE with PayGol Payment Gateway
This is crucial for the plugin to work and you to get paid.

Once you have signed up with paygol, go ahead and create a new service by choosing "Any Type Of Website".

When asked for "Type" choose "Integrated" and for the "Url Background (IPN) " (!Important) enter your-site-domain.com/paymobile-ipn. Now you go to My Services-> and copy & paste your ID into WP PayMobile Options. This should be a number and not a string.

That's it, enjoy your sales with this new monetization opportunity.
